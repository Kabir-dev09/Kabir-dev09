# Kabir

Student software developer building systems software and developer tools in Rust.

I'm building [Forge](https://github.com/Kabir-dev09/Forge), a Wayland-native terminal emulator with Vulkan rendering. It is my main project and where I work on terminal correctness, GPU text rendering, pane and tab management, configuration design, and performance measurement.

## Forge

- Stateful ANSI/VT processing with scrollback, selection, alternate screens, color, and resize/reflow
- GPU text rendering with glyph atlases, font fallback, ligature shaping, and procedural box-drawing and Braille glyphs
- Built-in tabs and pane management, including tiling and an experimental scrolling layout
- Event-driven rendering and benchmark harnesses for terminal, screen-buffer, and CPU tessellation paths
- TOML configuration with nested imports, validation, and safe defaults

Forge is under active development and currently targets Wayland on Linux. It still needs broader testing across compositors, graphics hardware, and drivers.
